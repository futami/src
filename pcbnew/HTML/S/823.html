  62      * Add non copper areas polygons (pads and tracks with clearance)

  63      * to a filled copper area

  64      * used in BuildFilledSolidAreasPolygons when calculating filled areas in a zone

  65      * Non copper areas are pads and track and their clearance area

  66      * The filled copper area must be computed before

  67      * BuildFilledSolidAreasPolygons() call this function just after creating the

  75 
  76     /**

  77      * Function buildThermalSpokes

  78      * Constructs a list of all thermal spokes for the given zone.

  79      */

  81                              std::deque<SHAPE_LINE_CHAIN>& aSpokes );
  82 
  83     /**

  84      * Build the filled solid areas polygons from zone outlines (stored in m_Poly)

  85      * The solid areas can be more than one on copper layers, and do not have holes

  86      *  ( holes are linked by overlapping segments to the main outline)

  87      * in order to have drawable (and plottable) filled polygons.

  88      * @return true if OK, false if the solid polygons cannot be built

  89      * @param aZone is the zone to fill

  90      * @param aRawPolys: A reference to a SHAPE_POLY_SET buffer to store

  91      * filled solid areas polygons (with holes)

  92      * @param aFinalPolys: A reference to a SHAPE_POLY_SET buffer to store polygons with no holes

  93      * (holes are linked to main outline by overlapping segments, and these polygons are shrinked

  94      * by aZone->GetMinThickness() / 2 to be drawn with a outline thickness = aZone->GetMinThickness()

  95      * aFinalPolys are polygons that will be drawn on screen and plotted

  96      */
  97     bool fillSingleZone( ZONE_CONTAINER* aZone, PCB_LAYER_ID aLayer, SHAPE_POLY_SET& aRawPolys,
  98                          SHAPE_POLY_SET& aFinalPolys );
  99 
 100     /**

 101      * for zones having the ZONE_FILL_MODE::ZONE_FILL_MODE::HATCH_PATTERN, create a grid pattern

 104      * @param aRawPolys: A reference to a SHAPE_POLY_SET buffer containing the initial

 107     void addHatchFillTypeOnZone( const ZONE_CONTAINER* aZone, PCB_LAYER_ID aLayer,

 121     // m_low_def can be used to define a low definition arc to polygon approximation
