  53 /**

  54  * SELECTION_TOOL

  55  *

  56  * Our sample selection tool: currently supports:

  57  * - pick single objects (click LMB)

  58  * - add objects to existing selection (Shift+LMB)

  59  * - draw selection box (drag LMB)

  60  * - handles MODULEs properly (i.e. selects either MODULE or its PADs, TEXTs, etc.)

  61  * - takes into account high-contrast & layer visibility settings

  62  * - invokes InteractiveEdit tool when user starts to drag selected items

  63  */

 171 private:
 172     /**

 173      * Function selectPoint()

 174      * Selects an item pointed by the parameter aWhere. If there is more than one item at that

 175      * place, there is a menu displayed that allows one to choose the item.

 176      *

 177      * @param aWhere is the place where the item should be selected.

 178      * @param aOnDrag indicates whether a drag operation is being performed.

 179      * @param aSelectionCancelledFlag allows the function to inform its caller that a selection

 180      * was cancelled (for instance, by clicking outside of the disambiguation menu).

 181      * @param aClientFilter allows the client to perform tool- or action-specific filtering.

 182      * @return True if an item was selected, false otherwise.

 183      */
 184     bool selectPoint( const VECTOR2I& aWhere, bool aOnDrag = false,
 185                       bool* aSelectionCancelledFlag = NULL,
 186                       CLIENT_SELECTION_FILTER aClientFilter = NULL );

 188     /**

 189      * Function selectCursor()

 192      * @param aForceSelect forces to select an item even if there is an item already selected.

 193      * @param aClientFilter allows the client to perform tool- or action-specific filtering.

 194      * @return true if eventually there is an item selected, false otherwise.

 195      */
 196     bool selectCursor( bool aForceSelect = false,
