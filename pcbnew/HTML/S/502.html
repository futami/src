 155     /**

 156      * Function biuParse

 157      * parses an ASCII decimal floating point value and scales it into a BIU

 158      * according to the current value of diskToBui.  This fuction is the complement of

 159      * fmtBIU().  One has to know what the other is doing.

 160      *

 161      * @param aValue is the ASCII value in C locale form with possible leading whitespace

 162      *

 163      * @param nptrptr may be NULL, but if not, then it tells where to put a

 164      *  pointer to the next unconsumed input text. See "man strtod" for more information.

 165      *

 166      * @return BIU - the converted Board Internal Unit.

 167      */

 170     /**

 175      * fmtDEG().  One has to know what the other is doing.

 176      *

 177      * @param aValue is the ASCII value in C locale form with possible leading whitespace

 178      *

 179      * @param nptrptr may be NULL, but if not, then it tells where to put a

 181      *

 182      * @return double - the string converted to a primitive double type

 183      */
