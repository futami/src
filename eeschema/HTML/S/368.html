  52 /**

  53  * A #SCH_PLUGIN derivation for loading schematic files created before the new s-expression

  54  * file format.

  55  *

  56  * The legacy parser and formatter attempt to be compatible with the legacy file format.

  57  * The original parser was very forgiving in that it would parse only part of a keyword.

  58  * So "$C", "$Co", and "$Com" could be used for "$Comp" and the old parser would allow

  59  * this.  This parser is not that forgiving and sticks to the legacy file format document.

  60  *

  61  * As with all SCH_PLUGINs there is no UI dependencies i.e. windowing calls allowed.

  62  */

  77         return wxT( "sch" );
