  73 public:
  74     /**

  75      * Set to true to synchronize pins at the same position when editing symbols with multiple

  76      * units or multiple body styles.  Deleting or moving pins will affect all pins at the same

  77      * location.

  78      * When units are interchangeable, synchronizing editing of pins is usually the best way,

  79      * because if units are interchangeable, it implies that all similar pins are at the same

  80      * location.

  81      * When units are not interchangeable, do not synchronize editing of pins, because each part

  82      * is specific, and there are no (or few) similar pins between units.

  83      *

  84      * Setting this to false allows editing each pin per part or body style regardless other

  85      * pins at the same location. This requires the user to open each part or body style to make

  86      * changes to the other pins at the same location.

  87      *

  88      * To know if others pins must be coupled when editing a pin, use SynchronizePins() instead

  89      * of m_syncPinEdit, because SynchronizePins() is more reliable (takes in account the fact

  90      * units are interchangeable, there are more than one unit).

  91      */
  92     bool          m_SyncPinEdit;
