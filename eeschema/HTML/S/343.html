  81 /**

  82  * A shim class between EDA_DRAW_FRAME and several derived classes:

  83  * LIB_EDIT_FRAME, LIB_VIEW_FRAME, and SCH_EDIT_FRAME, and it brings in a

  85  *

  86  * The motivation here is to switch onto GetScreen() for the underlying data model.

  88  * @author Dick Hollenbeck

  89  */

 149     /**

 150      * Function SelectComponentFromLib

 151      * Calls the library viewer to select component to import into schematic.

 152      * if the library viewer is currently running, it is closed and reopened

 153      * in modal mode.

 154      *

 155      * aAllowFields chooses whether or not features that permit the user to edit

 156      * fields (e.g. footprint selection) should be enabled. This should be false

 157      * when they would have no effect, for example loading a part into libedit.

 158      *

 159      * @param aFilter is a SCHLIB_FILTER filter to pass the allowed library names

 160      *  and/or the library name to load the component from and/or some other filter

 161      *          if NULL, no filtering.

 162      * @param aHistoryList       list of previously loaded components - will be edited

 163      * @param aUseLibBrowser     bool to call the library viewer to select the component

 164      * @param aUnit              preselected unit

 165      * @param aConvert           preselected De Morgan shape

 166      * @param aHighlight         name of component to highlight in the list.

 167      *                           highlights none if there isn't one by that name

 168      * @param aShowFootprints    whether to show footprints in the dialog

 169      * @param aAllowFields       whether to allow field editing in the dialog

 170      *

 171      * @return the selected component

 172      */
 173     COMPONENT_SELECTION SelectCompFromLibTree(
 174             const SCHLIB_FILTER* aFilter,
 175             std::vector<COMPONENT_SELECTION>& aHistoryList,
 176             bool aUseLibBrowser,
 177             int aUnit,
 178             int aConvert,
 179             bool aShowFootprints,
 180             const LIB_ID* aHighlight = nullptr,
 181             bool aAllowFields = true );
